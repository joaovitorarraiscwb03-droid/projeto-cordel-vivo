# Projeto Cordel Vivo na Escola — Static Export

Versão estática (HTML + CSS + imagens) pronta para hospedar.

## Deploy no Vercel (via GitHub)
1. Crie um repositório no GitHub e suba todos os arquivos desta pasta.
2. Em vercel.com → **Add New Project** → importe o repositório.
3. Framework Preset: **Other** (estático). Build Command: vazio. Output Directory: `.`
4. Deploy.

## Estrutura
- `index.html` — página completa
- `assets/` — imagens
- `vercel.json` — configuração mínima
